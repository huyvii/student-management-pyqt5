# 🎓 Student Management System (PyQt5)

Ứng dụng quản lý sinh viên đơn giản viết bằng **Python** và thư viện giao diện **PyQt5**. Hỗ trợ lưu trữ dữ liệu qua file CSV.

## ✨ Tính năng chính
* **Thêm sinh viên:** Nhập ID, Họ tên và GPA.
* **Hiển thị:** Danh sách sinh viên trực quan dưới dạng bảng.
* **Sửa/Xóa:** Cập nhật thông tin hoặc xóa sinh viên trực tiếp khỏi danh sách.
* **Lưu trữ:** Tự động lưu và đọc dữ liệu từ file `sv.csv`.

## 🛠 Công nghệ sử dụng
* **Ngôn ngữ:** Python 3.x
* **Giao diện:** PyQt5 (Thiết kế qua Qt Designer - file `.ui`)
* **Dữ liệu:** CSV (Comma Separated Values)
